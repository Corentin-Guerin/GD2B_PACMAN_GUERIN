Installation :
- Lancer l'application Mongoose present dans le dossier 
- Ouvrer un navigateur (Chrome de preference) 
- Entrer L'adresse suivant dans l'url : http://localhost:8000/


Controle : 
- Fleches directionnel / Pad Manette / ZQSD : Deplace PacMan
- F5 : Relance la partie 


Règles : 
- Récupérer toute les pieces sans vous faire toucher par les fantomes pour gagner
- En cas de contact avec les fantomes la game est perdu
